Project 96

A custom cursor
that follows you.

A precise dot paired with a trailing ring that eases toward your pointer, grows over interactive elements, and snaps magnetically to key targets. Move your mouse to feel it.

Try the playground How it works

Interactive targets

The cursor reacts differently depending on what it is over.

Hover to expand

Over any card the ring grows and softens, signalling an interactive surface.

Labelled hover

Some targets whisper a short label into the cursor as you approach them.

Magnetic pull

This card tugs the cursor toward its centre for a tactile, deliberate feel.

Magnetic playground

Buttons here lean into your pointer. Keyboard users get clean focus rings.

Built with care

  • Eased trailing via requestAnimationFrame and linear interpolation.
  • Touch aware — pointer devices only; touch keeps the native cursor.
  • Reduced motion — honours prefers-reduced-motion, dropping the trail.

Move, hover, click. The default cursor is hidden on pointer devices.
