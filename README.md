## About

Every project here is written in vanilla HTML, CSS, and JavaScript. No frameworks, no build tools, no npm, no bundlers. Each one lives in its own folder and opens by double-clicking `index.html`. The set ramps from static layouts through interactive components and small applications up to API-driven tools, so the early projects lock in structure and styling while the later ones cover state, the Canvas and Web Audio APIs, fetch, and browser platform features.

The design language stays consistent throughout: near-monochrome palettes with a single accent, strong type hierarchy, generous whitespace, semantic markup, keyboard support, and responsive layouts down to 375px. API projects use only free, no-auth endpoints and fall back to local mock data when an endpoint is unavailable.

## How to run

No install step and no server required.

```
git clone https://github.com/lazizbekravshanov/100-Frontend-Projects.git
cd 100-Frontend-Projects
```

Open any project by double-clicking its `index.html`, or from the terminal:

```
open "4. Pricing Cards - HTML & CSS/index.html"
```

A few later projects use browser platform APIs (Web Audio, Web Speech, MediaRecorder) that need a user gesture or a Chromium or Firefox based browser. The API projects fetch live data over the network and drop to local mock data when an endpoint is rate limited or offline.
