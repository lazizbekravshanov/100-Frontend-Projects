Layout patterns

The Holy Grail, built with Flexbox

A full-height page with a fixed header and footer, a left navigation rail, a fluid center column, and a supporting aside — all held together with nothing but flex containers.

How it works

The page is a vertical flex column that fills the viewport. The middle region grows to consume leftover space, then splits into three horizontal flex children. The center column takes flex: 1, so it expands while the side rails keep a fixed width.

Why it holds up

No absolute positioning, no floats, no fixed heights. Because the columns are flex siblings, they always match heights automatically, and the footer stays pinned to the bottom even when content is short.

Fluid center

Grows and shrinks with the viewport for comfortable reading widths.

Steady rails

Navigation and aside keep predictable widths on the desktop breakpoint.
