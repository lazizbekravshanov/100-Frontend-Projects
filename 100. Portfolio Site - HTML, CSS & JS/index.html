About

I’ve spent the last several years turning designs into resilient, accessible front ends — the kind that hold up on a five-year-old phone and read cleanly to a screen reader.

My approach is unglamorous on purpose: semantic HTML first, CSS that scales, and just enough JavaScript to make things feel alive. I sweat performance budgets, keyboard paths, and the difference between 1.4 and 1.6 line-height.
