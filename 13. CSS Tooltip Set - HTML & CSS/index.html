CSS Tooltip Set

Accessible, four-direction tooltips built entirely with data-attributes and pseudo-elements. Hover a trigger with the pointer, or tab to it with the keyboard — the tooltip appears either way.

Four directions

Each trigger declares its bubble text with data-tooltip and its placement with data-pos. Try the keyboard: press Tab to move focus and reveal each tooltip.

  • Top
  • Right
  • Bottom
  • Left

Inline in context

Tooltips work on any focusable element, not just buttons. Here they annotate words inside a paragraph. The is a lightweight way to explain a without cluttering the interface, and it stays fully reachable by .

How it works

  1. Add data-tooltip="Your text" to any focusable element.
  2. Set the placement with data-pos="top|right|bottom|left".
  3. A ::before renders the bubble; an ::after draws the arrow. Both fade in on :hover and :focus-visible. 
<button
  class="tooltip"
  data-pos="top"
  data-tooltip="Appears above">
  Hover / focus me
</button>
